# Support

Thanks for using Cella. Here's how to get help.

## Questions & help

- **GitHub Issues:** https://github.com/Awen-online/cella/issues , open an issue (use a "question" label if available).
- **General contact:** info@awen.online

## Reporting bugs

File a bug via Issues with steps to reproduce, expected vs. actual behavior, and your environment. See [CONTRIBUTING.md](CONTRIBUTING.md) for details.

## Security

Please do not report vulnerabilities publicly. See [SECURITY.md](SECURITY.md) , report privately to info@awen.online or via GitHub's private security advisory.

## Documentation

Start with the [README](README.md). Architecture and self-hosting docs land with the standalone migration (see the roadmap in [MAINTAINERS.md](MAINTAINERS.md)).
